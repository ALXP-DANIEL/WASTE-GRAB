# WasteGrab overview
- Nx monorepo for Angular + Node/Express.
- Apps: `shop` (Angular SSR storefront) and `api` (Node API with Docker support).
- Libraries: `libs/shop/*`, `libs/shared/models`, `libs/api/products`.
- Package manager: npm (package-lock.json present).
- Workspace uses Nx 22.7 and Angular 21.2.x.
